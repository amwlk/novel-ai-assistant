## 安全设计

### 认证流程
1. 用户登录 → 验证邮箱密码
2. 生成 JWT Token（有效期7天）
3. 前端存储 Token 到 localStorage
4. 请求时在 Header 中携带 Token
5. 后端验证 Token 有效性

### 数据安全
- 密码使用 bcryptjs 加密存储
- JWT Secret 配置在环境变量
- 数据库连接使用独立用户
- 敏感操作需要重新验证

## 部署架构

### 开发环境
```
本地开发:
- 前端: localhost:3000
- 后端: localhost:3002
- 数据库: Docker 容器
```

### 生产环境
```
Docker Compose:
- frontend: Next.js 容器
- backend: NestJS 容器
- postgres: PostgreSQL 容器
- redis: Redis 容器
- nginx: 反向代理
```
