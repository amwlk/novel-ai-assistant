# 长篇小说AI助手平台完整方案

**日期**：2026-03-19

## 一、项目概述

### 1. **项目定位**
- **纯工具平台**：专注于为网文作者提供AI辅助写作功能，无社交功能
- **核心价值**：帮助作者把控人物、时间线、物品、世界观、感情线，优化章节、节奏、故事发展走向、冲突、矛盾、高潮、结局
- **目标用户**：网文作者、小说创作者

### 2. **核心功能**
- **设定管理系统**：统一管理人物、世界观、道具等设定
- **双层知识库系统**：个人专属知识库 + 单本小说知识库
- **AI章节生成系统**：基于设定自动生成小说章节内容
- **智能提醒与矛盾检测**：检测设定矛盾，提供实时提醒
- **批量生成与任务队列**：使用BullMQ实现异步章节生成
- **可视化图谱**：人物关系、势力层级、感情线时间轴可视化

## 二、技术架构

### 1. **前端架构**
- **框架**：React 18 + TypeScript
- **UI库**：Ant Design 5.x
- **状态管理**：Redux Toolkit + Redux Saga
- **路由**：React Router 6.x
- **构建工具**：Vite 4.x
- **样式**：CSS Modules + Tailwind CSS

### 2. **后端架构**
- **框架**：NestJS 10.x + TypeScript
- **数据库**：PostgreSQL 15 + TypeORM
- **缓存**：Redis 7.x
- **消息队列**：BullMQ 5.x
- **API文档**：Swagger 6.x
- **认证**：JWT + Passport.js

### 3. **AI架构**
- **框架**：Python 3.10 + PyTorch 2.x
- **模型**：Doubao-LLM 2.0
- **向量数据库**：Pinecone 2.x
- **嵌入模型**：text-embedding-ada-002
- **API网关**：FastAPI 0.100.x
