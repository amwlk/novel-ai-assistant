## 第一部分：项目概述与核心定位

### 1.1 项目核心定位

**平台本质**：专为网文创作者打造的AI创作提效中台

**核心价值主张**：
1. **设定统一管理**：12大弹性要素体系，修改一处自动同步
2. **智能知识库**：双层知识库（个人专属+单本小说），AI全程调用
3. **创作流程闭环**：要素→知识库→大纲→细纲→章节→优化
4. **纯工具定位**：无社交、无社区、数据完全私有

**目标用户群体**：
- 核心用户：日更作者（50-100万人）
- 扩展用户：业余作者（200-300万人）
- 潜在用户：网文爱好者（1000万人以上）

**平台定位边界**：
- ✅ 允许：设定管理、AI生成、创作辅助、数据导出
- ❌ 禁止：社交功能、用户互动、社区分享、公开展示

### 1.2 核心痛点解决方案

#### 痛点1：设定管理混乱
**现状**：
- 人物设定分散在Excel
- 世界观记录在Word
- 道具清单在记事本
- 灵感想法在备忘录

**解决方案**：
- 统一的12大弹性要素管理系统
- 所有设定集中存储，自动关联
- 修改一处，全局同步
- 实时版本控制

#### 痛点2：长篇连贯性差
**现状**：
- 写到后面忘记前面设定
- 人物性格前后不一致（OOC）
- 道具设定忘记
- 时间线混乱

**解决方案**：
- 双层知识库系统自动沉淀
- AI生成时实时调用知识库
- 智能矛盾检测与提醒
- 版本回溯与对比

#### 痛点3：创作效率低下
**现状**：
- 日更压力大（每天6000字）
- 卡文频率高（每周2-3次）
- 修改耗时（每章30-60分钟）
- 有效创作时间短

**解决方案**：
- AI辅助章节生成
- 批量生成与任务队列
- 智能卡文续写
- 一键去AI味润色

#### 痛点4：AI不懂作品
**现状**：
- 每次都要重新交代设定
- AI生成内容风格不符
- 需要学习复杂提示词
- 成本控制困难

**解决方案**：
- 专属知识库自动管理
- AI上下文智能记忆
- 网文专属提示词模板
- 个性化训练与适配

### 1.3 核心功能体系

#### 12大弹性要素体系

**基础核心要素（7个）**：
1. 大纲：整体故事线（起承转合、核心爽点）
2. 世界观规则：世界背景、力量体系、社会规则
3. 人物设定：主角/配角/反派的外貌、性格、背景
4. 组织势力：宗门/家族/公司/阵营设定
5. 势力关系：各势力间的敌对/结盟/利用关系
6. 主线支线剧情：分卷/分阶段的剧情铺陈
7. 章节细纲：单章的核心情节、对话节点

**题材扩展要素（3个）**：
8. 物品道具：武器/法宝/信物/关键物品设定
9. 场景设定：关键场景的环境描述、氛围
10. 背景资料：历史背景、文化背景、隐藏设定

**自定义要素（2个）**：
11. 地点场景：关键地理位置、场景分布
12. 自定义要素：作者自定义的特殊要素

**设计原则**：
- 可填充、可调用：每个元素都是独立编辑模块
- 有逻辑、无强制：按创作逻辑排序，不强制顺序
- 精细化、大容量：支持数千字深度描述
- 可联动、可更新：修改一处自动同步关联模块

### 1.4 双层知识库系统

#### 个人专属知识库
**存储内容**：
- 作者风格偏好（常用词、句式、节奏）
- 常用设定模板（人物、世界观、道具）
- 历史作品设定（跨作品复用）
- 个性化训练数据（作者专属模型）

**使用策略**：
- 权重：30%（个人）:70%（单本）
- 检索：关键词匹配 + 语义相似度
- 更新：作者手动更新 + AI自动学习

#### 单本小说知识库
**存储内容**：
- 12大要素的精细化内容
- 已生成的章节内容
- 创作过程中的细节记录
- 版本历史与变更

**使用策略**：
- 权重：70%（单本）:30%（个人）
- 检索：标签匹配 + 上下文理解
- 更新：实时自动沉淀

**知识库调用策略**：
1. **标签匹配**：所有内容自动打标签
2. **上下文理解**：分析当前创作场景
3. **关联权重排序**：核心要素权重最高
4. **语义检索**：向量数据库支持语义匹配

### 2.2 技术选型详细说明

#### 前端技术栈

**React 18 + TypeScript**
- 类型安全：TypeScript提供编译时类型检查
- 组件化：React组件化开发，提高复用性
- 性能优化：React 18并发特性提升性能
- 生态成熟：丰富的第三方库支持

**Next.js 14**
- SSR支持：服务端渲染提升SEO和首屏速度
- 路由系统：文件系统路由，开发体验好
- API路由：前后端一体化开发
- 图片优化：自动优化图片加载

**Tailwind CSS + shadcn/ui**
- 原子化CSS：快速开发，样式一致
- 组件库：shadcn/ui提供高质量组件
- 主题系统：支持暗色模式和主题切换

- 数据可视化：D3.js提供强大的可视化能力
- 交互图谱：React Flow提供拖拽、缩放
- 性能优化：虚拟化渲染大图谱

**Zustand**
- 状态管理：轻量级状态管理库

### 2.4 架构设计原则

#### 模块化设计
- **职责分离**：每个模块职责明确
- **高内聚低耦合**：模块内部高内聚，模块间低耦合
- **可复用性**：通用功能抽象为共享模块
- **可测试性**：每个模块可独立测试

#### 渐进式架构
- **初期单体**：MVP阶段采用单体架构
- **逐步拆分**：后期根据需要逐步拆分为微服务
- **预留接口**：为微服务化预留接口
- **平滑演进**：架构演进不影响现有功能

#### 性能优化
- **缓存策略**：多层缓存（Redis、浏览器缓存）
- **异步处理**：耗时操作异步化（任务队列）
- **懒加载**：前端路由懒加载
- **虚拟化**：大列表虚拟化渲染

// backend/src/modules/knowledge/knowledge.service.ts
import { Injectable } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { KnowledgeBase } from './knowledge.entity';
import { EmbeddingService } from './embedding.service';
import { AIService } from '../ai/ai.service';

@Injectable()
export class KnowledgeService {
  constructor(
    @InjectRepository(KnowledgeBase)
    private readonly knowledgeRepository: Repository<KnowledgeBase>,
    private readonly embeddingService: EmbeddingService,
    private readonly aiService: AIService,
  ) {}

  async addContent(
    userId: string,
    novelId: string,
    contentType: string,
    contentId: string,
    content: string,
    tags: string[],
  ): Promise<KnowledgeBase> {
    // 生成向量嵌入
    const embeddings = await this.embeddingService.generateEmbedding(content);

    const knowledge = this.knowledgeRepository.create({
      userId,
      novelId,
      contentType,
      contentId,
      content,
      tags,
      embeddings,
    });

    return await this.knowledgeRepository.save(knowledge);
  }

  async search(
    userId: string,
    novelId: string,
    query: string,
    topK: number = 10,
  ): Promise<KnowledgeBase[]> {
    // 生成查询向量
    const queryEmbedding = await this.embeddingService.generateEmbedding(query);

    // 相似性搜索
    const results = await this.knowledgeRepository
      .createQueryBuilder('knowledge')
      .where('knowledge.user_id = :userId', { userId })
      .andWhere('knowledge.novel_id = :novelId', { novelId })
      .andWhere('knowledge.is_active = TRUE')
      .orderBy('knowledge.embeddings <-> :queryEmbedding', 'DESC')
      .setParameter('queryEmbedding', queryEmbedding)
      .limit(topK)
      .getMany();

    return results;
  }

  async getNovelContext(novelId: string, maxTokens: number = 100000): Promise<string> {
    // 获取小说的知识库内容
    const knowledge = await this.knowledgeRepository.find({
      where: { novelId, is_active: true },
      order: { createdAt: 'ASC' },
    });

    // 构建上下文
    let context = '';
    let tokenCount = 0;

    for (const item of knowledge) {
      const itemContext = this.formatContextItem(item);
      const itemTokens = this.estimateTokens(itemContext);

      if (tokenCount + itemTokens > maxTokens) {
        break;
      }

      context += itemContext + '\n\n';
      tokenCount += itemTokens;
    }

    return context;
  }

  private formatContextItem(item: KnowledgeBase): string {
    const typeMap = {
      element: '设定要素',
      chapter: '章节内容',
      note: '创作笔记',
    };

    return `【${typeMap[item.contentType] || item.contentType}】
标签: ${item.tags?.join(', ') || '无'}
内容: ${item.content}`;
  }

  private estimateTokens(text: string): number {
    // 简单估算：1个中文字符 ≈ 1.5个token
    return Math.ceil(text.length * 1.5);
  }
}

  ) {
    this.connection = new Redis({
      host: process.env.REDIS_HOST || 'localhost',
      port: parseInt(process.env.REDIS_PORT || '6379'),
      password: process.env.REDIS_PASSWORD,
    });

    this.chapterQueue = new Queue('chapter-generation', {
      connection: this.connection,
    });
  }

  onModuleInit() {
    // 启动Worker处理章节生成任务
    new Worker(
      'chapter-generation',
      async (job: Job) => {
        await this.processChapterJob(job);
      },
      {
        connection: this.connection,
        concurrency: 5, // 同时处理5个任务
      },
    );
  }

  async addChapterGenerationJob(
    novelId: string,
    chapterIds: string[],
    userId: string,
  ): Promise<string> {
    const job = await this.chapterQueue.add(
      'generate-chapters',
      {
        novelId,
        chapterIds,
        userId,
      },
      {
        priority: 1, // 优先级1-10
        attempts: 3, // 最多重试3次
        backoff: {
          type: 'exponential',
          delay: 1000, // 指数退避
        },
      },
    );

    return job.id;
  }

  async addBatchGenerationJob(
    novelId: string,
    chapterCount: number,
    startNumber: number,
    userId: string,
  ): Promise<string> {
    const job = await this.chapterQueue.add(
      'batch-generate-chapters',
      {
        novelId,
        chapterCount,
        startNumber,
        userId,
      },
      {
        priority: 2,
        attempts: 3,
        backoff: { type: 'exponential', delay: 2000 },
      },
    );

    return job.id;
  }

  async getJobStatus(jobId: string): Promise<any> {
    const job = await this.chapterQueue.getJob(jobId);
    if (!job) return null;

    return {
      jobId: job.id,
      status: await job.getState(),
      progress: await job.progress(),
      timestamp: job.timestamp,
    };
  }

  private async processChapterJob(job: Job): Promise<void> {
    const { novelId, chapterIds, userId } = job.data;

    for (let i = 0; i < chapterIds.length; i++) {
      const chapterId = chapterIds[i];
      
      try {
        // 获取章节信息
        const chapter = await this.chapterService.getChapter(chapterId);
        
        // 生成章节内容
        const content = await this.aiService.generateChapter(
          novelId,
          chapter.chapterNumber,
          chapter.title,
          chapter.outline?.prompt || '',
          chapter.wordCount || 2500,
        );

        // 保存章节内容
        await this.chapterService.updateChapterContent(chapterId, content);

        // 更新任务进度
        await job.updateProgress(((i + 1) / chapterIds.length) * 100);
      } catch (error) {
        console.error(`生成章节 ${chapterId} 失败:`, error);
        
        if (job.attemptsMade < job.opts.attempts) {
          throw error; // 重试
        }
        
        // 记录失败
        await this.chapterService.markChapterFailed(chapterId, error.message);
      }
    }
  }

  async processBatchGenerationJob(job: Job): Promise<void> {
    const { novelId, chapterCount, startNumber, userId } = job.data;

    for (let i = 0; i < chapterCount; i++) {
      const chapterNumber = startNumber + i;
      
      try {
        // 创建章节
        const chapter = await this.chapterService.createChapter(
          novelId,
          chapterNumber,
          `第${chapterNumber}章`,
          2500,
          userId,
        );

        // 生成章节内容
        const content = await this.aiService.generateChapter(
          novelId,
          chapterNumber,
          chapter.title,
          '',
          chapter.wordCount,
        );

        // 保存章节内容
        await this.chapterService.updateChapterContent(chapter.id, content);

        // 更新任务进度
        await job.updateProgress(((i + 1) / chapterCount) * 100);
      } catch (error) {
        console.error(`生成章节 ${chapterNumber} 失败:`, error);
        
        if (job.attemptsMade < job.opts.attempts) {
          throw error;
        }
      }
    }
  }

  async cleanupCompletedJobs(days: number = 7): Promise<void> {
    const cutoffDate = new Date();
    cutoffDate.setDate(cutoffDate.getDate() - days);

    await this.chapterQueue.clean(0, days);
  }
}

// frontend/src/components/visualization/CharacterRelationGraph.tsx
import React, { useEffect, useRef, useState } from 'react';
import * as d3 from 'd3';
import { CharacterNode, CharacterLink } from '@/types/visualization';

interface CharacterRelationGraphProps {
  nodes: CharacterNode[];
  links: CharacterLink[];
  onNodeClick?: (node: CharacterNode) => void;
  onLinkClick?: (link: CharacterLink) => void;
}

export const CharacterRelationGraph: React.FC<CharacterRelationGraphProps> = ({
  nodes,
  links,
  onNodeClick,
  onLinkClick,
}) => {
  const svgRef = useRef<SVGSVGElement>(null);
  const [width, setWidth] = useState(0);
  const [height, setHeight] = useState(500);

  useEffect(() => {
    if (!svgRef.current) return;

    const container = svgRef.current.parentElement;
    if (container) {
      setWidth(container.clientWidth);
    }

    const handleResize = () => {
      if (container) {
        setWidth(container.clientWidth);
      }
    };

    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  useEffect(() => {
    if (!svgRef.current || width === 0) return;

    drawGraph();
  }, [nodes, links, width]);

  const drawGraph = () => {
    const svg = d3.select(svgRef.current);
    svg.selectAll('*').remove();

    const simulation = d3.forceSimulation(nodes)
      .force('link', d3.forceLink(links).id((d: any) => d.id).distance(150))
      .force('charge', d3.forceManyBody().strength(-400))
      .force('center', d3.forceCenter(width / 2, height / 2))
      .force('collide', d3.forceCollide().radius(60));

    // 绘制连线
    const link = svg.append('g')
      .selectAll('line')
      .data(links)
      .join('line')
      .attr('stroke', (d: CharacterLink) => d.color || '#94a3b8')
      .attr('stroke-width', 2)
      .attr('opacity', 0.6);

    // 绘制节点
    const node = svg.append('g')
      .selectAll('circle')
      .data(nodes)
      .join('circle')
      .attr('r', 25)
      .attr('fill', (d: CharacterNode) => {
        switch (d.type) {
          case 'protagonist': return '#f97316'; // 橙色
          case 'antagonist': return '#ef4444'; // 红色
          case 'support': return '#3b82f6'; // 蓝色
          default: return '#64748b'; // 灰色
        }
      })
      .attr('stroke', '#fff')
      .attr('stroke-width', 3)
      .on('click', (event: MouseEvent, d: CharacterNode) => {
        event.stopPropagation();
        onNodeClick?.(d);
      })
      .call(d3.drag<SVGCircleElement, CharacterNode>()
        .on('start', dragstarted)
        .on('drag', dragged)
        .on('end', dragended));

    // 节点标签
    const label = svg.append('g')
      .selectAll('text')
      .data(nodes)
      .join('text')
      .text((d: CharacterNode) => d.name)
      .attr('text-anchor', 'middle')
      .attr('fill', '#fff')
      .attr('font-size', '12px')
      .attr('font-weight', 'bold')
      .attr('dy', '0.35em');

    // 力导向图模拟
    simulation.on('tick', () => {
      link
        .attr('x1', (d: any) => d.source.x)
        .attr('y1', (d: any) => d.source.y)
        .attr('x2', (d: any) => d.target.x)
        .attr('y2', (d: any) => d.target.y);

      node
        .attr('cx', (d: any) => d.x)
        .attr('cy', (d: any) => d.y);

      label
        .attr('x', (d: any) => d.x)
        .attr('y', (d: any) => d.y);
    });

    function dragstarted(event: d3.DragEvent<SVGCircleElement, CharacterNode, any>) {
      if (!event.active) simulation.alphaTarget(0.3).restart();
      event.subject.fx = event.subject.x;
      event.subject.fy = event.subject.y;
    }

    function dragged(event: d3.DragEvent<SVGCircleElement, CharacterNode, any>) {
      event.subject.fx = event.x;
      event.subject.fy = event.y;
    }

    function dragended(event: d3.DragEvent<SVGCircleElement, CharacterNode, any>) {
      if (!event.active) simulation.alphaTarget(0);
      event.subject.fx = null;
      event.subject.fy = null;
    }
  };

  return (
    <div className="character-relation-graph">
      <svg ref={svgRef} width={width} height={height} />
    </div>
  );
};

import { Injectable } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { Chapter } from './chapter.entity';
import { CreateChapterDto, UpdateChapterDto } from './dto';
import { NovelService } from '../novel/novel.service';

@Injectable()
export class ChapterService {
  constructor(
    @InjectRepository(Chapter)
    private readonly chapterRepository: Repository<Chapter>,
    private readonly novelService: NovelService,
  ) {}

  async createChapter(
    novelId: string,
    chapterNumber: number,
    title: string,
    wordCount: number,
    userId: string,
  ): Promise<Chapter> {
    const chapter = this.chapterRepository.create({
      novelId,
      chapterNumber,
      title,
      wordCount,
      userId,
    });

    const savedChapter = await this.chapterRepository.save(chapter);

    // 更新小说的章节计数
    await this.novelService.updateChaptersCount(novelId, 1);

    return savedChapter;
  }

  async updateChapterContent(
    chapterId: string,
    content: string,
  ): Promise<Chapter> {
    const chapter = await this.chapterRepository.findOne({
      where: { id: chapterId },
    });

    if (!chapter) {
      throw new Error('章节不存在');
    }

    // 计算字数
    const wordCount = this.calculateWordCount(content);

    Object.assign(chapter, {
      content,
      wordCount,
      status: 'draft',
      updatedAt: new Date(),
    });

    const savedChapter = await this.chapterRepository.save(chapter);

    // 更新小说的总字数
    const wordsToAdd = wordCount - (chapter.wordCount || 0);
    await this.novelService.updateWordsCount(chapter.novelId, wordsToAdd);

    return savedChapter;
  }

  async updateChapter(
    chapterId: string,
    dto: UpdateChapterDto,
  ): Promise<Chapter> {
    const chapter = await this.chapterRepository.findOne({
      where: { id: chapterId },
    });

    if (!chapter) {
      throw new Error('章节不存在');
    }

    Object.assign(chapter, dto);
    return await this.chapterRepository.save(chapter);
  }

  async deleteChapter(chapterId: string): Promise<void> {
    const chapter = await this.chapterRepository.findOne({
      where: { id: chapterId },
    });

    if (!chapter) {
      throw new Error('章节不存在');
    }

    const novelId = chapter.novelId;
    const wordCount = chapter.wordCount || 0;

    await this.chapterRepository.remove(chapter);

    // 更新小说的章节和字数计数
    await this.novelService.updateChaptersCount(novelId, -1);
    await this.novelService.updateWordsCount(novelId, -wordCount);
  }

  async getChapter(chapterId: string): Promise<Chapter> {
    return await this.chapterRepository.findOne({
      where: { id: chapterId },
    });
  }

  async getChapters(novelId: string): Promise<Chapter[]> {
    return await this.chapterRepository.find({
      where: { novelId },
      order: { chapterNumber: 'ASC' },
    });
  }

  async getNextChapterNumber(novelId: string): Promise<number> {
    const lastChapter = await this.chapterRepository.findOne({
      where: { novelId },
      order: { chapterNumber: 'DESC' },
    });

    return (lastChapter?.chapterNumber || 0) + 1;
  }

  private calculateWordCount(text: string): number {
    // 简单计算：中文字符数
    return text.replace(/[^\u4e00-\u9fa5]/g, '').length;
  }
}
