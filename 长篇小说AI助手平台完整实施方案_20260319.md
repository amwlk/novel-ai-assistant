## 一、项目概述与核心定位

### 1.1 项目本质与定位

**平台定位**: 专为网文创作者打造的AI创作提效中台

**核心价值主张**:
- 解决网文作者三大核心痛点：设定管理混乱、长篇崩设定、创作效率低下
- 提供"懂作品的AI"：基于作者专属知识库的智能辅助
- 纯工具定位：无社交、无分享、数据完全私有

**目标用户群体**:
- 核心用户：日更作者（50-100万人）
- 扩展用户：业余作者（200-300万人）
- 潜在用户：网文爱好者（1000万人以上）

### 1.2 12大弹性要素体系（核心骨架）

#### 第一层：基础核心要素（6个，所有题材通用）
1. **大纲**：小说整体故事线（起承转合、核心爽点/虐点、关键剧情节点）
2. **世界观规则**：世界背景、力量体系、社会规则、货币/等级体系
3. **人物设定**：主角/配角/反派的外貌、性格、背景、执念、成长线、口头禅
4. **组织势力**：宗门/家族/公司/阵营等势力设定
5. **势力关系**：各势力间的敌对/结盟/利用等关系
6. **主线支线剧情**：分卷/分阶段的剧情铺陈，支线与主线衔接点

#### 第二层：题材扩展要素（按题材分类）
- **玄幻/仙侠**：势力体系、修炼等级、丹药/法宝体系
- **现代言情**：感情线、人物羁绊、家庭背景
- **悬疑/推理**：伏笔线索、案件脉络、嫌疑人设定
- **都市爽文**：金手指规则、逆袭节点、反派阵营
- **科幻/星际**：科技体系、星际势力、种族设定
- **历史穿越**：时代背景、历史人物、朝代规则

#### 第三层：自定义要素（作者按需创建，无数量限制）

#### 设计原则：
- **可填充、可调用**：每个元素都是独立编辑模块，支持手动写、AI生成、反复修改
- **有逻辑、无强制**：按创作逻辑排序，仅做创作引导，不强制按顺序生成
- **精细化、大容量**：突破短句限制，支持单元素数千字深度描述
- **可联动、可更新**：修改任一元素，AI自动同步到所有关联模块

### 1.3 双层知识库系统（核心灵魂）

#### 个人专属知识库
- 存储作者固定创作风格、常用设定、个人快捷指令
- 跨作品复用，建立作者个人创作品牌
- 支持作者专属模型训练数据

#### 单本小说知识库
- 存储该小说的12大弹性要素、大纲/细纲、已生成章节
- 随创作动态更新，形成活的知识库
- AI全程调用，保证内容一致性

#### 联动逻辑
- AI生成时同时调取个人知识库（风格/常用设定）+小说知识库（情节/设定）
- 知识库自动沉淀所有创作内容，无需手动整理
- 支持关键词检索、版本回溯、设定校验

### 1.4 四位一体创作流程

```
要素 → 知识库 → 大纲 → 细纲 → 章节
```

**流程说明**：
1. **要素搭建**：填充12大弹性要素，形成创作骨架
2. **知识库构建**：AI自动沉淀所有设定，形成专属知识库
3. **大纲设计**：AI生成总大纲→分卷大纲→章节细纲
4. **章节生成**：批量生成章节→人工轻量介入→智能润色

### 2.4 架构设计原则

#### 1. 渐进式架构
- **起步阶段**：单体应用，快速验证
- **成长阶段**：模块化拆分，预留微服务接口
- **成熟阶段**：微服务架构，独立扩展

#### 2. 模块化设计
- **高内聚低耦合**：每个模块职责明确
- **接口抽象**：定义清晰的模块接口
- **依赖注入**：使用NestJS依赖注入机制

#### 3. 可扩展性设计
- **插件化AI提供者**：支持动态添加AI模型
- **配置化业务规则**：业务规则通过配置管理
- **事件驱动架构**：使用事件总线解耦模块

#### 4. 性能优化设计
- **缓存策略**：Redis缓存热点数据
- **异步处理**：耗时操作异步化
- **CDN加速**：静态资源CDN分发
- **数据库优化**：索引优化、查询优化

#### 向量检索设计

```typescript
// 向量数据库设计
interface VectorDocument {
  id: string;
  novelId: string;
  type: DocumentType;
  content: string;
  embedding: number[]; // 1536维向量（text-embedding-ada-002）
  metadata: {
    source: string;
    version: number;
    tags: string[];
    createdAt: number;
  };
}

// 向量检索服务
class VectorSearchService {
  async search(
    query: string,
    novelId: string,
    topK: number = 5
  ): Promise<VectorResult[]> {
    const queryEmbedding = await this.embed(query);
    return await this.vectorDB.similaritySearch(
      queryEmbedding,
      novelId,
      topK
    );
  }
  
  async getRelated(
    documentId: string,
    topK: number = 5
```

```typescript
export class ChapterGeneratorService {
  async generateChapter(
    novelId: string,
    chapterNumber: number,
    prompt: string,
    options: GenerationOptions
  ): Promise<GenerationResult> {
    // 1. 构建上下文
    const context = await this.buildContext(novelId, chapterNumber);
    
    // 2. 构建提示词
    const fullPrompt = this.buildPrompt(context, prompt, options);
    
    // 3. 调用AI模型
    const rawResult = await this.aiProvider.generate(fullPrompt);
    
    // 4. 后处理
    const processedResult = this.postProcess(rawResult);
    
    // 5. 返回结果
    return {
      content: processedResult.content,
      wordCount: processedResult.wordCount,
      generationTime: processedResult.generationTime,
      model: processedResult.model,
      cost: processedResult.cost
    };
  }
  
  async batchGenerate(
    novelId: string,
    startChapter: number,
    endChapter: number,
    prompt: string
  ): Promise<BatchGenerationResult> {
    // 创建批量任务
    const taskId = await this.queue.add({
      type: 'batch-generation',
      novelId,
      startChapter,
      endChapter,
      prompt
    });
    
    return {
      taskId,
      total: endChapter - startChapter + 1,
      status: 'queued'
    };
  }
}
```

```typescript
    try {
      const result = await this.chapterGenerator.generateChapter(
        novelId,
```

```typescript
@Injectable()
export class ResultPostProcessorService {
  async postProcess(
    rawContent: string,
    options: PostProcessOptions = {}
  ): Promise<PostProcessedResult> {
    let processedContent = rawContent;
    
    // 1. 去除AI痕迹
    if (options.removeAIMarkers) {
      processedContent = this.removeAIMarkers(processedContent);
    }
    
    // 2. 语言润色
    if (options.polish) {
      processedContent = await this.polishLanguage(processedContent);
    }
    
    // 3. 格式化
    if (options.format) {
      processedContent = this.formatContent(processedContent);
    }
    
    // 4. 敏感词检测
    const sensitiveWords = this.detectSensitiveWords(processedContent);
    
    return {
      content: processedContent,
      sensitiveWords,
      wordCount: this.countWords(processedContent),
      processingTime: Date.now() - options.startTime
    };
  }
  
  private removeAIMarkers(text: string): string {
    // 移除AI特有的表达方式
    const patterns = [
      /作为AI/g,
      /我是一个AI/g,
      /我无法/g,
      /我不能/g,
      /请告诉我/g,
      /请提供/g
    ];
    
    let result = text;
    for (const pattern of patterns) {
      result = result.replace(pattern, '');
    }
    
    return result;
  }
  
  private polishLanguage(text: string): string {
    // 使用AI进行语言润色
    return this.aiProvider.polish(text, {
      style: 'natural',
      tone: 'novel'
    });
  }
  
  private formatContent(text: string): string {
    // 格式化为网文格式
    // 1. 段落分隔
    // 2. 标点符号标准化
    // 3. 引号标准化
    // 4. 数字格式化
    
    let result = text;
    
    // 段落分隔
    result = result.replace(/\n{3,}/g, '\n\n');
    
    // 标点符号标准化
    result = result.replace(/[，。、！？：；“”‘’（）]/g, (match) => {
      const map: Record<string, string> = {
        '，': '，',
        '。': '。',
        '、': '、',
        '！': '！',
        '？': '？',
        '：': '：',
        '；': '；',
        '“': '“',
        '”': '”',
        '‘': '‘',
        '’': '’',
        '（': '（',
        '）': '）'
      };
      return map[match] || match;
    });
    
    return result;
  }
  
  private detectSensitiveWords(text: string): string[] {
    // 加载敏感词库
    const sensitiveWords = fs.readFileSync(
      path.join(__dirname, 'sensitive-words.txt'),
      'utf-8'
    ).split('\n').filter(w => w.trim());
    
    // 检测敏感词
    const foundWords = sensitiveWords.filter(word => 
      text.includes(word)
    );
    
    return foundWords;
  }
  
  private countWords(text: string): number {
    // 统计字数（中文字符）
    const chineseChars = text.match(/[\u4e00-\u9fa5]/g);
    return chineseChars ? chineseChars.length : 0;
  }
```
